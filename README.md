 
# About
This is a Microsoft Teams clone built with React JS and Chat Engine!

React JS is the programming language of choice, and Chat Engine is a set of APIs and UI Kit specific to chat.

## Built-with
  - ReactJs
  - CSS custom properties
  - CSS Grid
  - Flexbox
  - JSX
